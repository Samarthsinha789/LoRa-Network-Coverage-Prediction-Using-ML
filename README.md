# LoRa Network Coverage Prediction using Machine Learning

## Overview
This project predicts LoRa network coverage quality using Machine Learning models trained on datasets generated from MATLAB simulations.

## Technologies Used
- Python
- MATLAB
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Models
- Random Forest
- Decision Tree
- Linear Regression

## Dataset
Generated using MATLAB simulations with:
- Distance
- Spreading Factor
- Bandwidth
- Transmit Power
- Environment Type

## Results
- Compared multiple ML models.
- Predicted coverage as Good, Moderate and Poor.
- Visualized results using heatmaps and confusion matrices.

## Future Scope
Improve prediction using real-world LoRa gateway data and deep learning models.
